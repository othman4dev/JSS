What is JSS?

JSS is a JavaScript library for building user interfaces. It is declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called “components”.

#div2

Why learn JSS?

JSS will help you write JavaScript in a CSS-like syntax, which will give you a better control of the DOM and save you a lot of time.

How powefull ?
